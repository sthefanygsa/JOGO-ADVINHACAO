📌 Jogo de Adivinhação - Descrição 
<p>
  Este é um jogo simples de adivinhação desenvolvido com HTML, CSS e JavaScript. O objetivo é acertar um número aleatório entre 1 e 100 dentro de um número limitado de tentativas.
A cada palpite, o jogo informa se o número correto é maior ou menor, ajudando o jogador a se aproximar da resposta. Caso acerte, uma mensagem de sucesso é exibida; se as tentativas acabarem, o número correto é revelado.
O projeto foi criado com foco em praticar conceitos básicos de lógica de programação, manipulação do DOM e interação com o usuário.
</p>
